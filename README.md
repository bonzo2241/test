# Учебный сайт с тестом

## Запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите приложение:
   ```bash
   python app.py
   ```

После запуска приложение доступно по адресу `http://localhost:5000`.

## Учётные записи

- Администратор создаётся автоматически:
  - логин: `admin`
  - пароль: `admin`

Обычных пользователей можно зарегистрировать через форму.
