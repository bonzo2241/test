# Многоагентная система поддержки обучения

Платформа на Flask и SPADE для поддержки обучения в разных предметных областях.
В ней реализованы агенты мониторинга, диагностики, рекомендаций и генерации черновиков комментариев.

По умолчанию проект поставляется с демонстрационным набором тем и тестов по истории
в файле `content/seed_content.json`, но код приложения не привязан к конкретному предмету.
Можно заменить этот файл на любой другой предметный набор без изменения Python-кода.

## Запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите приложение:
   ```bash
   python app.py
   ```

После запуска приложение доступно по адресу `http://localhost:5000`.

## Учётные записи

- Администратор (преподаватель) создаётся автоматически:
  - логин: `admin`
  - пароль: `admin`

Обычных пользователей можно зарегистрировать через форму.

## SPADE-агенты

Код агентов находится в `spade_agents.py`. Для запуска нужен XMPP‑сервер.
Укажите адреса и пароли в переменных окружения и запустите:

```bash
python spade_agents.py
```

Агенты используют те же данные и логи активности, что и веб‑приложение.


## Онтологии

Проект поддерживает предметную онтологию через секцию `ontology` в `content/seed_content.json`.
Онтология описывает концепты и связи `prerequisites`, которые используются агентом рекомендаций
для подбора опорных разделов при ошибках.

Администратор может открыть экспорт онтологии в формате Turtle по адресу `/ontology`.
